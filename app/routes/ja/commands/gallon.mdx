## 必殺技

### ビーストキャノン

オーラをまとって突進するガロンの代表的な必殺技。特にES版は象徴と言っても差し支えないだろう。

突っ込んだ後にレバーを任意の方向に入れてPボタンで方向転換できる。方向転換後は再び相手を攻撃することが可能。通常版だと最大2ヒット、ESで出すと最大5ヒットする。

3種類の出し方があるが、初段の軌道以外はすべて同じ性能。小中大による違いもない。

#### 地上（ES）

```
↓↘→ + P
```

この技は [キャラ紹介](../characters/gallon) でも取り上げているので、まずはそちらを。

基本的にはESを使う。通常版でもコンボはつながるので同時押しをミスしてもリスクは小さい。

ESを出したつもりで上方に方向転換してしまうと相手の近くで隙だらけになって流石にまずい。うっかり通常版になってしまった場合は

- 大人しく後ろに引き返す
- そのまま着地してしれっと投げにいく

あたりでフォローする。空中で反撃される分には大きなダメージはまず喰らわないので落ち着いて。

通常版でもコンボのダメージは悪くないので、ゲージがない時のフィニッシュ用としては有用。それ以外の場面でも使えるが、ダウンさせて起き攻めに行った方がリターンが見込める。

#### 空中（ES）

```
空中で ↓↘→ + P
```

これもまずは [キャラ紹介](../characters/gallon) をどうぞ。

地上とは逆に基本的には通常版で出して、方向転換はしない。当たれば着地後にESクライムレイザーで拾えるので、これをESで出す必要性は薄い。

ダメージはクライムコンボよりもESビースト単体の方が大きいので、絶対に当たる自信があるなら出してもいいが例外的。

#### 対空（ES）（GC）

```
→↓↘ + P
```

扱いが難しいので無理に使う必要はない。

対空とは言うが、発生が遅いので相手の飛び込みを落とすのには使えない。ジャンプで逃げようとする相手を捕まえたり、空中から仕掛けようとしたところにカウンターで当てる。レイレイのレバー大Pと同じイメージ。

斜め上に飛んでいくため、方向転換しないと隙だらけの状態で落ちてくるので注意。

GC技としては発生が遅いので反撃性能は高くないが、移動技としての能力は高い。普通に出した時と同じように方向転換もできる。

### クライムレイザー（ES）

```
↓↑ + K（溜め不要）
```

通称「サマー」だが無敵はない。その代わりに溜めも不要。空中ビーストからのコンボでESを使うことが多い。

発生が早く、出際は空中ガード不能で足元に喰らい判定がないなど性能自体は高く、暴れや起き攻め拒否の選択肢として意外に有効な場面がある。その場合もリターンを重視してESがメインで使われる。

### ミリオンフリッカー（ES）

```
↓↙← + P
```

ヌンチャクを振り回す技。通常版は小中大で踏み込み距離や判定が変わったり、ボタン連打でヒット数が増加するが使わない。

ESは密着で出すとたくさん削れるので、フィニッシュ狙いで使うことがある。起き攻めで生で使ってもいいし、めくり小K→小足→ESミリオンフリッカーで当たってもガードされてもOKで出してもいい。

### ワイルドサーキュラー（ES）

```
相手の近くで →↘↓↙← + 中K or 大K
```

これも [キャラ紹介](../characters/gallon) で紹介しているので、ここでは特に触れない。

## EX必殺技

### ドラゴンキャノン

```
←↙↓↘→ + KK
```

画面外から炎の龍を呼び出して相手を攻撃する技。主な用途は削りだが、相手の行動によっては抜けられる場合もある。

発生してしまえばガロン本体がダメージを受けても消えないので、フィニッシュ時に相打ちOKで使ってもいい。

ダークガロン（キャラ選択時にスタート+PPP or KKKで決定）で使うと炎のグラフィックが変わり、性能は低下する。発生が僅かに遅くなり、出した後の隙が大きい。

### モーメントスライス

```
小P・中P・→・小K・中K
```

順押し系コマンドなのであらゆる技をキャンセルして出せる。チェーンをキャンセルしてのコンボで使う。

コマンドの関係で素直に入力するだけで小P→中Pのチェーンからコンボがつながる。屈小Pスタートで下段にするといい。

ESビーストよりも全体のダメージは少ないが、以下のメリットがある。

- 赤ダメージが多い
- 当てた後に小クライムx2でゲージを溜められる
- ガードされても有利

起き攻めを仕掛ける時にクイックムーブで裏周りから狙うと極めてガードされにくい。

チェーンは屈中P→中足にできるとダメージが更に上るが、入力の難しさはそれ以上に跳ね上がる。

1. 中指（屈中P）
2. 親指（中K）・人差し指（小P）・中指（中P）
3. レバーを前に入れる
4. 人差し指（小K）・中指（中K）

これを素早く入力しつつ、1と2がチェーンコンボになるようタイミングを調整する。
