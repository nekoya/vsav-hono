## 必殺技

### ディオ=セーガ（ES）

```
地上 or 空中で ↓↘→ + P
```

設置型の飛び道具で、地上でも空中でも出せる。通常版は3ヒット、ES版は5ヒットする。

この技の使い方は [キャラ紹介](../characters/jedah) で触れているので、そちらでどうぞ。

### ネロ=ファティカ（ES）

```
↓↙← + P
```

爪痕を残しながら後退して、相手を捕まえたら追撃する技。実は打撃ではなく投げ技だが相手が空中にいても拾える。決めた後は追い打ちが確定する。

見た目は対空っぽいが発生が遅くて判定も弱いので、相手の飛びを見てから出しても基本的に負ける。先読みで出して本体が相手の技が当たらないところまで下がればどうにか使える程度。

ESセーガを空中ガードさせると相手がこちら側に飛んでくるので、そこをESで捕まえるのに使うぐらいが現実的。それもタイミングが悪いと一方的に殴られるので注意。

地味に飛び道具を相殺できる。

### イラ=スピンタ（ES）

```
空中で →↘↓↙← + K
```

空中で少し静止した後、急降下して相手を投げる技。小中大で下降する角度が深くなる（前に出る）。ESは大より前に出て、ダメージも大きい。

相手を画面端に追い込んで大かESで出すのが定石。ジャンプからでもいいし、ダッシュから直接でも、D中Kなどをキャンセルしてでも出せる。ガードを固めている相手への奇襲として使える。

降下中は無敵でも何でもないので対空などで落とされる時は落とされるが、着地の隙は小さいので空中で一発もらうだけ。奇襲のために背負うリスクとしては悪くはない。空中にいる間に反撃しようと飛んできた相手を捕まえると気分がいい。

### イラ=ピアノ（ES）

```
イラ=スピンタの予備動作中に P
```

イラ=スピンタの予備動作（空中で静止）中にPボタンを追加入力すると出せる打撃技。使わない。

発生がイラ=スピンタよりも遅いので、対空しようとした相手に対してズラしとして当たることもなくはない（CPU戦で身に覚えがあるはず）が、威力も大したことないのでわざわざ狙う必要はない。

空中にいる間にどうにか反撃しようと飛んできた相手に落とされることも増えるので、使わなくていい技。

### スプレジオ（ES）（GC）

```
ガード中に →↓↘ + P
```

GC専用技。自分の首をかき切って、そこから出る血しぶき（正確には血ではないが便宜上そう記述する）で攻撃する。

ESだと最大3ヒットするが、カス当たりになると相手がダウンしなくて反撃をもらうことがある。なので使うなら通常版が基本だが、それでも性能はよくない。

そもそもの発生が遅くGCしても相手のガードが間に合ったり、攻撃判定も高い位置にしか出ないので背の低いキャラなどには当たらないことも多い。

使うならそれなりの覚悟を持って。

### サングェ=パッサーレ

```
相手の近くで →↘↓↙← + 中K or 大K
```

コマンド投げ。CPUがやたら決めてくるので錯覚しがちだが、発生が遅くて無敵もないので基本的には使わない。ES版は存在しない。

中下段に対してガードを固めている相手に対して空ダッシュから狙えなくもないが、小足と発生が同じなのでダメージが高くて相手の打撃と相打ちが取れるスカし下段の方が期待値は高い。

## EX必殺技

### フィナーレ=ロッソ

```
↓↓ + PP
```

相手の足元に血の池を展開して沈め、地下でタコ殴りにする技。当てた後は追い打ちが確定する。

発生は早くはないが、相手が飛び道具を出そうとした時にカウンターとして合わせたり、遠くで隙の大きな技やDFを解除された時など狙いどころは意外にある。

セーガで相手の動きを抑え込んで使うのも有効。ジェダは地味にCPU戦が難しいので、この連携にはお世話になる。対戦でも使えるが、相手がジャンプしてセーガに当たるとそこで終わってしまうのに注意。

見てから慌てて垂直ジャンプすると着地をきれいに吸えたり、画面端でのバックジャンプを捕まえたりもできる。「これもある」と思わせることで相手の意識配分に負荷をかけるだけでも価値はある。

天雷破をガードした後の反撃にも使えるが、距離が近いと上から降ってくる鉄球を喰らってしまう。

### プロヴァディ=セルヴォ

```
←↙↓↘→ + KK の後に K
```

手首を切って、流れる血から手を伸ばして相手を捕まえ契約書に判を付かせる通称「契約」。投げ判定なので、相手が投げられない状態だとつかめない。

Kボタンを追加入力した時に手を伸ばすので、距離によって押すタイミングを変える必要がある。つかみ判定自体は広いのでタイミングはシビアではないが、近すぎると連打してもつかめない。

あまり使う技ではないが、ダメージは大きいので小技の刻みから相手が忘れた頃に出すのはあり。天雷破への反撃にも使える。

この技でフィニッシュした後に大Pで勝ちポーズを決めると、ちょうど契約書にジェダがもたれかかるように見える。
