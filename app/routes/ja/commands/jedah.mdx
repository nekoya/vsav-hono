## 必殺技

### ディオ=セーガ（ES）

```
地上 or 空中で ↓↘→ + P
```

### ネロ=ファティカ（ES）

```
↓↙← + P
```

### イラ=スピンタ（ES）

```
空中で →↘↓↙← + K
```

### イラ=ピアノ（ES）

```
イラ=スピンタの予備動作中に P
```

### スプレジオ（ES）（GC）

```
ガード中に →↓↘ + P
```

### サングェ=パッサーレ

```
相手の近くで →↘↓↙← + 中K or 大K
```
