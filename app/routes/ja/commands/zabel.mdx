## 必殺技

### デスハリケーン（ES）

```
地上 or 空中で ↓↙← + K
```

### スカルスティング（ES）

```
地上 or 空中で ↓↑ + K （溜め不要）
```

### デスフレーズ（ES）（GC）

```
ガード中に →↓↘ + K
```

### ヘルズゲート

```
←↙↓↘→ + K
```

### スカルパニッシュ（ES）

```
相手の近くで →↘↓↙← + 中P or 大P
```
