## 必殺技

### ソウルフラッシュ（ES）

```
地上 or 空中で ↓↘→ + P
```

#### 地上版

射程距離の短い飛び道具。言ってみれば我道拳だが、性能は悪くない。

小中大で弾速は違うが飛距離は同じ。つまり速い大はすぐに消え、遅い小は長く残る。ESは小と同じ弾速でより遠くまで飛び、3ヒットする。

相手を画面端に追い込んで小 or ESソウルを盾にしてジャンプやダッシュで攻め込むのが強い。

通常技のラッシュに混ぜると発生の遅さが逆にズラしとして有効に作用して、AG入力しようとした相手に当たればそのままコンボにつなげられる。ESは通常版よりも発生が遅い分だけズラし効果も高くなる。

ESソウルの利点は他にもあり飛距離も長く3ヒットする上に、発生後はリリス本体がダメージを受けても消えない。極めて優秀な壁となる。発生の速い中足はもちろん、ジャンプ・ダッシュ攻撃と絡めたコンボでの大ダメージが狙える。

起き攻めに置いておくのも有効。ESソウルを盾にして飛び込むと、相手のジャンプや暴れを抑止できるので一方的にジャンプ攻撃とスカし下段の二択を迫れる。そこからのチェーンルミナスが破壊力抜群。

#### 空中ソウルフラッシュ

基本的な性質は地上版と同じで、出した高さで前に飛ぶ。飛距離こそ短いものの、空中から入ってくる相手に対する壁として効果は高い。特に空中ダッシュしてくる相手は基本的に空中ガードできないので、見えていても突っ込んで当たってしまうことが多い。

デミトリのカオスフレアと違って、どのボタンで出しても空中での反動は同じ。あまり後ろには下がらず、そのまま着地する。

壁として使う場合は画面上に長く残したいので小で出す。ゲージに余力があれば無理に突っ込んでくる相手に相打ちOKでESを置いてもいいが効率はよくない。ESを使うなら地上版と同じように、画面端での攻めのパーツとして使う方が有効。

#### 低空ソウルフラッシュ

地上にいる時に↓↘→↗ + Pでジャンプしてすぐに空中ソウルフラッシュを出すと、普通に出すよりも低い位置から出せる。これも基本的には小で出す。

ジャンプから入力するより早く出せる上に、頭の高さぐらいと位置もいい。防御手段として非常に優秀なので別項目として取り上げたい。

<blockquote class="twitter-tweet" data-media-max-width="560"><p lang="ja" dir="ltr">セイヴァー、リリスの低空ソウル入力サンプルです。↓から↑の入力が速いとハイジャンプになるので、斜め下も使ってゆっくりレバーを回すのがコツ。ボタンも早く押すと地上版になるので、全体的にゆっくり落ち着いて入力しましょう。 <a href="https://t.co/crHw6uSkwE">pic.twitter.com/crHw6uSkwE</a></p>&mdash; nekoya (@nekoya) <a href="https://twitter.com/nekoya/status/1885545745190969720?ref_src=twsrc%5Etfw">February 1, 2025</a></blockquote> <script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

必要な入力は↓↘→↗ + Pだが、素直に入力するとハイジャンプ（↓↑、斜め入力可）が出てしまうのでコツがいる。

ハイジャンプは立ち状態から一定時間以内に下方向→上方向と入力すると出るので、その入力受付時間が切れるよう下方向に長めに入れる。少しの間しゃがんでから出すか、ゆっくりめに↙↓↘→とレバーを回すと暴発を防げる。

ボタンも押すのが早すぎるとレバーを↗に入れていても地上で出てしまう。全体的に落ち着いてゆっくり入力するのがコツ。

これを安定して出せるようになると、リリスの立ち回りの安定感は大きく増す。

一方で、失敗してハイジャンプが出てしまっても慌てず、そのまま攻め続ける図太さも重要。

### シャイニングブレイド（ES）（GC）

```
→↓↘ + P
```

昇竜拳だが性能はそこまで高くない。小中大で飛び上がる高さが変わり、前進する距離も少し伸びる。全体のモーションが短い小だけ使えばいい。

デミトリのデモンクレイドルやモリガンのシャドウブレイドに比べて、

- 発生が遅い（ESは同じ）
- 無敵が切れた後の判定が弱い

といった欠点があるため過信は禁物。

![](https://pbs.twimg.com/media/GijF-TTaAAA92iu?format=jpg&name=small)

地上で攻撃判定が発生した次のフレームには無敵が切れる。頭部に喰らい判定があるので、相手の飛び込みと相打ちになりやすい。

![](https://pbs.twimg.com/media/GijF-TRbcAA3xcO?format=jpg&name=small)

飛び上がった後は攻撃判定を包むように喰らい判定が出現するのに加えて下半身は無防備。相手を翼の上に乗せる感覚で使う必要がある。

遠めの牽制に合わせたり、GCを出しても一方的に潰されることも多い。

ESは発生も速く飛び上がるまで全身無敵、しかもGCで出した場合は [ガード不能ESGC](../techniques/unblockable_esgc) になり高性能だが、3回飛び上がるのが余計。

相手の伸ばした手足を刈るように出した場合、1発目だけが当たって相手が転んでしまう。2・3発目を空振りしている間に相手が起き上がってきて、着地の隙にお釣りをもらってしまう。

ESシャイニングを出す時は必ず相手の身体に当てること。

### メリーターン（ES）

```
↓↙← + K
```

回転しながら飛び上がり、翼で斬りつける技。多段ヒットするES版が最後の削りに使える。通常版は使わない。足元無敵はあるが、だからといって特に役に立つほどではない。

ビシャモン、ビクトル、アナカリス、ジェダ、デミトリ、オルバスには近くから出せば相手がしゃがんでいても多段ヒットする。オルバスには3回、それより高いキャラは4回当たるので、ガードされても結構な削りダメージを取れる。詳しくは [各キャラの当たり判定](../statistics/hitBox) を参照。

それ以外のキャラであっても、相手がダッシュ攻撃を立ちガードする場合は有効。

### ミスティックアロー（ES）

```
→↘↓↙← + P
```

遠くからでも出せるコマンド投げ。相手に飛びついて、つかんだら翼を弓にして相手を矢として発射する。

小中大の順で遠くに飛ぶ。発生までの時間はすべて同じ。相手との距離に合わせて出す必要があり、遠くはもちろん近すぎてもつかめない。

相手を画面端に追い詰めて出せば距離の調整は不要だが、逆側に投げるため画面端から逃がしてしまう。

ESは大と同じ軌道で飛び、壁に当たった相手がその場で落ちずに跳ね返ってくるところを空中で追撃できる。

基本的にはルミナスを当てておけばいいが、グルーミーパペットショウなど好きな技でどうぞ。距離が近ければESシャイニングが出しやすくてダメージも高い。

画面端に追い詰めてアローを出した場合は距離が遠くなるので注意。ルミナスは届くが早めに出す必要がある。

## EX必殺技

### ルミナスイリュージョン

```
地上 or 空中で 小Px2・→・小K・大P
```

俗に言う瞬獄殺。空中でも出せる。順押し系コマンドなので、あらゆる通常技をキャンセルできる。チェーンコンボからこの技につないで大ダメージを奪うのがリリスの真骨頂。

昇竜はモリガンよりも性能が低いリリスだが、この技は本家の完全上位互換。

まず目に付くのはダメージの高さ。単体での攻撃値もさることながら、D中Kから入ることが多いモリガンに対して、J大P（背の高いキャラにはJ大P→大K）から始動のコンボが主体のリリスはより火力が出しやすい。ダクネスより段数が少ないのも補正の面で有利に働く。

空中ルミナスは性能が高く、予備動作1フレーム（空中ダクネスは3フレーム）で全身無敵。攻撃判定の発生と同時に無敵はなくなるが、それでも空中で出せる必殺技としては破格の性能。空中ガード後の反撃はもちろん、空中チェーンに対する割り込みにも使える。

### スプレンダーラブ

```
→↓↘ + KK
```

脱衣して空中に飛び上がる技。使わなくていい。

一応、ESアローからうまく巻き込めば、最大でルミナスやESシャイニングよりも減ることはある。

### グルーミーパペットショウ

```
←↙↓↘→ + KK
```

放り投げたシルクハットが当たると音ゲーが始まり、そのスコアに応じたダメージが入る。2ゲージ消費するだけの価値はないので使わなくていい。

ESアローの追撃で入るが、距離が遠いと届かないので相手を画面端に追い詰めた時には使えない。それ以外で狙うとすれば相手の空中ダッシュに合わせるぐらいだが、発生が遅くて2ゲージ使うのでリスクには見合わない。

譜面は6パターンあり、出現率がそれぞれ異なる。難しい譜面は出現率が低いので基本パターンをある程度練習してあとは運任せにしてもいい。
