## 必殺技

### ビッグブレス（ES）

```
↓↘→ + P
```

鼻息を吹き出す飛び道具。立大Kを [空キャン](../techniques/kara_cancel) で出してゲージ溜めするのが主な用途だが、意外と空中から入ってくる相手への壁にもなる。

小中大で見た目は変わらないが判定は小が近く、大が遠くに出る。そこまで大きな違いはないので、あまり気にしなくていい。

### ビッグブロウ（ES）

```
→↓↘ + P
```

腕を振り回してパンチを繰り出す突進技。使わない。立中Pキャンセルビッグタイフーンの暴発で見る。

Pボタンを押しっぱなしにしていると溜めを作ってしまう。バレッタのカゴと同じく有用性はないので、暴発で出てしまったとしても溜めないように気をつけたい。

### ビッグタイフーン（ES）（GC）

```
→↓↘ + K
```

足を回転させながら蹴りを出す技。主な使い道は立中PキャンセルでESを出してコンボにすること。ショートダッシュ攻撃からのコンボは通称「バンバンチーン」。たった3段でめちゃくちゃ減る。

通常版は当たっても相手がダウンせず、多段技で当たり方も安定しないので主には同時押しのミスで見ることになる。

GC技にもなっており、ゲージを節約しつつ切り返したい場面では通常版を使うこともある。小は発生が早いが喰らい判定が前に出ているのと当ててフレーム不利なので、中を使っておく方がリターンが見込める。ただしガードされると隙が大きいので、確実に当てられる場面で使おう。

### ビッグタワーズ（ES）

```
↓↓ + P
```

氷柱を出して相手を攻撃する技。小中大の順で攻撃が出ている時間が長くなる。隙は変わらないので、意外と大で出すのもあり。

空中ガード可能だが、しゃがんだ姿勢で出るため判定は強め。ちゃんと出せば、相手のジャンプ攻撃に対して相打ち以上は取れる場合が多い。

すぐ出せるコマンドではないが空中から入ってくる相手に対しては選択肢として持っておいていい技。特にジェダに対して有効。

ESは移動する氷柱を前後に出す。速くて高性能な飛び道具にもなるし、相打ち狙いの対空としても使える。相手が技を出していると2ヒットする場合があり、一瞬ですごいダメージ（しかも大半が赤）が入ることがある。

正確には飛び道具ではないので、相手の飛び道具と干渉しない。発生も攻撃も速くて相手の飛び道具をすり抜けてダメージも高いので、相手が飛び道具を出そうとしているところに合わせるのも強い。

強力だが、それゆえにゲージが余ったサスカッチが出したくなる技でもある。苦し紛れに出して上を飛ばれないように注意。

### ビッグブランチ（ES）

```
→↘↓↙← + P
```

相手を飲み込み、凍らせて吐き出すコマンド投げ。無敵はないが予備動作が1フレームで、決めた後にバンバンチンを入れると体力ゲージを半分ぐらい持っていける、リターンが極めて大きい技。

凍結時間はキャラによって異なる上にレバガチャで短縮できるので確定はしないが、回復した後にそのままショートダッシュ攻撃が当たってしまうことも少なくない。相手が速く抜けてくるようなら、ESで決めると凍結が通常版よりも長くなるのでコンボが確定する。

間合いの外や相手が投げられない状態でも発動するので、垂直ジャンプで避けられると逆に痛いコンボをもらってしまうので注意。

狙い所は [キャラ紹介](../characters/sasquatch) で触れているので、そちらを参照。地味に移動起きからリバーサルで出せる（セイヴァーでは普通は移動起きからコマンド投げは出せない）ので一発逆転も狙える。

### ビッグスイング（ES）

```
レバー一回転 + K
```

こちらもコマンド投げ。基本的な性能はビッグブランチと同じで、あちらの方が追撃のリターンがあるためほぼ使わない。

ゲージがなくて相手の残り体力が少ない場合には、追撃不要で単発のダメージが大きいこちらで倒し切るのには使える。

## EX必殺技

### ビッグフリーザー

```
←↙↓↘→ + PP
```

氷のレーザーを吐き出す技。発生は遅いが、相手の飛び道具をかき消して当てられるのでカウンターとしては悪くない。

ESタワーズと同じタイミングで出すと、相手がタワーズにタイミングを合わせていた場合にズラしになって当たる場合がある。

反動で少し後ろに下がるので、遠めの相手の飛び込みに相打ちを狙って出したら一方的に勝ってしまうこともある。

意外とあなどれない技。

### ビッグアイスバーン

```
←↙↓↘→ + KK
```

地面に氷を張って、相手がその上にいると池に落とす技。近すぎると当たらないが、かなり遠くまで届く。

発生は早くないが、ガードに集中していると意識から外れるので意外に回避は難しい。画面端だと見てからバックジャンプで逃げても着地で踏んでしまう。

使わなくても十分に強いサスカッチだが、無視できるダメージでもないので「これもある」と相手の意識に入れるだけでも十分に強い。

### ビッグトラップ

```
←↙↓↘→ + スタートボタン
```

バナナを食べて、その皮を地面に投げ捨てる技。スタートボタンを使う珍しい技で、実質的に挑発だがダメージは僅かにある。

基本的にはガードできないが、相手が攻撃をしゃがみガード中に踏んだ場合だけはガードできる。実戦で起きるのは、サスカッチの攻撃をガードしている途中で踏んだ場合。ただしその場合でも立ちガードだと踏んで滑って転んでしまう。

### ビッグスレッジ

```
レバー二回転 + KK
```

ビッグスイングの強化版と考えていい。実用性で言うならESブランチでいい。
